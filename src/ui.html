What does the plugin?

It allows designers to incorporate real data into their designs by connecting to a database and populating their designs with up-to-date product data.

How to use the plugin?

In the settings tab first you have to fill in the credentials to connect to the database.

  • Database link
  • Project API key
  • Source API key

Once connection is made the annotations will be drawn immediately on a new layer in the figma project. You can toggle the annotation view with the button on the bottom left.

In the connect panel you find everything to create / update / delete an annotation.

To create a new annotation you simply select a node in your Figma design and fill in the input fields for the node. When everything is filled in you can press enter or push the create annotation button.

To update an annotation it is almost the same as creating a new annotation but now you select a node with an already existing annotation and you can edit the fields you want and you can press enter or push the update annotation button to update the annotation.

To delete an existing annotation you select the node of which you want the annotation deleted and you click on the remove annotation button in the connect panel. Or if you want to delete a node completely you can delete the node completely and the annotation linked to the node will also be deleted.
