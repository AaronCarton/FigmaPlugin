Welcome to the Propertize plugin by
delaware

What does the plugin?

It allows designers to incorporate real data into their designs by connecting to a database and populating their designs with up-to-date product data.

How to use the plugin?

In the settings tab, you have to fill in the credentials first before connecting to the database.

  • Database url
  • Project API key
  • Source API key

Once connection is made, the annotations will be drawn immediately on a new layer in the Figma project. You can toggle the annotation's visibility with the button on the bottom left.

In the connect panel you find everything to create / update / delete an annotation.

To create a new annotation you simply select a node in your Figma design and fill in the input fields for the node. When everything is filled in you can press enter or push the create annotation button.

To update an annotation it is almost the same as creating a new annotation. First, you select a node with an alreay existing annotation and edit the values in the form. When you are done editing, you can press enter or push the update annotation button to update the annotation.

To delete an existing annotation you select the node of which you want the annotation deleted; When the node is selected, you click on the remove annotation button in the connect panel. Or if you delete a node in the Figma design that has an annotation linked to it, the annotation will be removed aswell.
